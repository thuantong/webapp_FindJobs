Trạng thái:
- Bài tuyển dụng:
		+ 0 : Chờ duyệt
		+ 1 : Đã được duyệt
		+ 2 : Bị từ chối (Kèm tin nhắn)
		+ 3 : Đã chỉnh sửa
- Duyệt tin :
		+ 0 : Chưa đọc
		+ 1 : Đã đọc
		+ 2 : Đã duyệt
		+ 3 : Đã từ chối

- Giới tính :
		+ 1 : Nam
		+ 2 : Nữ
		+ 3 : Tất cả 

- Tài Khoản[status]:
		+ 0 : Ngoại tuyến
		+ 1 : Đang hoạt động
		+ 2 : Đã tạm khóa
