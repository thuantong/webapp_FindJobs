		+ 0 : Ngoại tuyến
		+ 1 : Đang hoạt động
		+ 2 : Tạm khóa
- Bài tuyển dụng:
		+ 0 : Chờ duyệt
		+ 1 : Đã được duyệt
		+ 2 : Bị từ chối (Kèm tin nhắn)
		+ 3 : Đã chỉnh sửa
4: đã tạm dừng
- Duyệt tin :
		+ 0 : Chờ duyệt
		+ 1 : Đã được duyệt
		+ 2 : Đã bị từ chối
		

- Giới tính :
		+ 1 : Nam
		+ 2 : Nữ
		+ 3 : Tất cả 

- Tài Khoản[status]:
		+ 0 : Ngoại tuyến
		+ 1 : Đang hoạt động
		+ 2 : Đã tạm khóa
		+ 3 : Chưa xác nhận
- Đơn xin việc :
	+ Người tìm việc
		+ 0 : Mới ứng cử
		+ 1 : Cần xác nhận phỏng vấn
		+ 2 : Phỏng vấn
		+ 3 : Đã bị từ chối
		+ 4 : Trúng tuyển
		+ 5 : Rớt
	+ Nhà tuyển dụng
		+ 0 : Mới ứng cử
		+ 1 : Chờ xác nhận phỏng vấn
		+ 2 : Phỏng vấn
		+ 3 : Đã từ chối
		+ 4 : Trúng tuyển
		+ 5 : Đã loại
